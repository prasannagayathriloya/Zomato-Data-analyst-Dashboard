# Zomato-Data-analyst-Dashboard
Zomato Data Analysis Dashboard project focused on analyzing restaurant trends, ratings, and customer preferences using Excel, Power BI, Tableau, and SQL Queries


## Project Objective
To analyze Zomato restaurant data and understand trends in ratings, cost, and customer preferences.

## Key Insights
- Most restaurants have average ratings between 3.0 and 4.0
- Online delivery is more popular in metro cities
- Cost varies significantly across countries
